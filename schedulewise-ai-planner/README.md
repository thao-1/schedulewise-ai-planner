# ScheduleWise

ScheduleWise is an AI-powered schedule management application designed to optimize your productivity and work-life balance. The app intelligently creates personalized schedules based on your preferences and habits.

## About the App

ScheduleWise helps users:
- Create optimized weekly schedules based on personal preferences
- Balance work, personal activities, and rest time
- Track productivity metrics and work-life balance
- Integrate with Google Calendar for seamless schedule syncing

The app was deployed using Vercel. App link: https://schedulewise-sigma.vercel.app/

## Key Features

- AI-generated schedules tailored to your work style
- Deep work optimization
- Work-life balance tracking
- Google Calendar integration
- Customizable preferences
- Responsive design for all devices

## Technologies Used

- **Frontend**: React with TypeScript
- **UI Framework**: Shadcn UI components
- **State Management**: React Context API
- **Routing**: React Router
- **API Integration**: React Query
- **Authentication**: Supabase Auth
- **Backend**: Supabase Functions (Deno)
- **AI Integration**: OpenAI API
- **Styling**: Tailwind CSS
- **Deployment**: Vercel

## Development Practices

- Modern React patterns with functional components and hooks
- TypeScript for type safety
- Responsive design principles
- Code quality tools: ESLint, Prettier, Trunk
